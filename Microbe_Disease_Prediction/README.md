# Description

Benchmarking the methods of KATZHMDA, NBLPIHMDA, BiRWMP, BPNNHMDA, NTSHMDA, HMDA-Pred and LRLSHMDA for the Microbe-Disease Associations (MDsAs) prediction.


# Usage
Please see readme.txt in each subdirectory.
